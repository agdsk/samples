## Code Samples

These folders contain sample selections from projects I've worked on.

### Demo-Booker

**Laravel 5.2**

An application that allowed consumers to book appointment times for a hands-on demonstration of a virtual reality
headset at retail stores in the US and EU. Brand ambassadors met with the customer to provide a guided tour of the
product. 

### Mailing-Lists

**CodeIgniter**

Extracts from a web application that produces automotive customer marketing mailing lists for dealerships.

### Training-Center

**Symfony 3.0**

The online class scheduling system for an American Heart Association training center operated by a large hospital
network. Hospital employees registered for required certification training courses and teachers conducted classes and
certification exams.

### Twilio-cleaner

**Symfony Command Line**

PHP 8 utility to scan a database of phone numbers against Twilio error records with multithreaded processing.
